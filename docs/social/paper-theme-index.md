# Paper → Theme Index

The drafting + reply-matching lookup table. When a news item or a stranger's
post touches a theme, find it below and reach for that paper.

- **Link format:** `https://federalistreader.org/papers/{n}`
- **Card:** attach `social-cards/{n}.jpg`
- **Authorship:** consensus attribution; "disputed" = long argued between
  Hamilton and Madison. When in doubt, say "Publius."
