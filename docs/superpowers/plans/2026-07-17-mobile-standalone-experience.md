# Mobile Standalone Experience Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Give Federalist Reader an `F` broadside Home Screen identity, a polished standalone launch experience, and a composed two-row reading toolbar on phones narrower than 480px.

**Architecture:** A build-time icon generator owns one outlined vector master and exports every committed raster size plus the matching favicon. Static manifest and Apple metadata progressively enhance the shared Astro shell, while display-mode and width media queries adapt the existing layout without introducing a service worker or installed-only application layer. The current preferences module and toolbar script remain the state authority.

**Tech Stack:** Astro 7 static output, TypeScript, CSS, `opentype.js`, `sharp` 0.35.3 as a build-only dev dependency, Vitest 4, and Playwright 1.61.

## Global Constraints

- The installed identity is the approved newsprint broadside mark with a double ink rule, oxblood `F`, and restrained printer's ornaments.
- Installed names are `Federalist Reader` and the short name `Federalist`.
- Manifest `start_url` and `scope` are `/`; `display` is `standalone`.
- No service worker, offline cache, update prompt, custom Apple launch-image matrix, account, synchronization, or sticky mobile dock.
- The paper remains a full-width single-column broadside on phones.
- At widths below 30rem (480px), modes/read status occupy row one and text size spans row two.
- Every interactive toolbar target remains at least 44 × 44 CSS pixels.
- Unread uses a transparent oxblood check control; read uses an oxblood-filled control with a light check.
- Metadata failure or unsupported standalone behavior must leave the complete server-rendered website usable.
- Do not stage or modify the unrelated untracked `.claude/` directory.

- [ ] **Step 4: Implement the vector master and deterministic exports**

Create `scripts/generate-app-icons.mjs`:

```js
import { mkdir, writeFile } from 'node:fs/promises';
import { fileURLToPath } from 'node:url';

import opentype from 'opentype.js';
import sharp from 'sharp';

const NEWSprint = '#e7deca';
const INK = '#2a2722';
const OXBLOOD = '#672b28';
const SIZE = 512;

export function buildAppIconSvg(font) {
  const initial = font.getPath('F', 0, 0, 286);
  const bounds = initial.getBoundingBox();
  const x = (SIZE - (bounds.x2 - bounds.x1)) / 2 - bounds.x1;
  const y = (SIZE + (bounds.y2 - bounds.y1)) / 2 - bounds.y1 + 3;
  const path = font.getPath('F', x, y, 286).toPathData(2);

  return `<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 ${SIZE} ${SIZE}" data-app-icon="federalist-reader" data-monogram="F">
  <rect width="512" height="512" fill="${NEWSprint}"/>
  <rect x="44" y="44" width="424" height="424" fill="none" stroke="${INK}" stroke-width="10"/>
  <rect x="60" y="60" width="392" height="392" fill="none" stroke="${INK}" stroke-width="4"/>
  <g fill="${OXBLOOD}"><path d="${path}"/></g>
  <g fill="${INK}" stroke="${INK}" stroke-width="4">
    <path d="M170 116H342M170 396H342"/>
    <path d="M256 103l13 13-13 13-13-13zM256 383l13 13-13 13-13-13z" fill="${OXBLOOD}" stroke="none"/>
  </g>
</svg>`;
}

async function generate() {
  const fontPath = new URL(
    '../node_modules/@fontsource/libre-caslon-display/files/libre-caslon-display-latin-400-normal.woff',
    import.meta.url,
  );
  const font = await opentype.load(fileURLToPath(fontPath));
  const svg = buildAppIconSvg(font);
  const svgBuffer = Buffer.from(svg);
  const iconsDirectory = new URL('../public/icons/', import.meta.url);

  await mkdir(iconsDirectory, { recursive: true });
  await writeFile(new URL('../public/app-icon.svg', import.meta.url), `${svg}\n`);
  await writeFile(new URL('../public/favicon.svg', import.meta.url), `${svg}\n`);
  await sharp(svgBuffer).resize(180, 180).png().toFile(fileURLToPath(new URL('../public/apple-touch-icon.png', import.meta.url)));
  await sharp(svgBuffer).resize(192, 192).png().toFile(fileURLToPath(new URL('../public/icons/icon-192.png', import.meta.url)));
  await sharp(svgBuffer).resize(512, 512).png().toFile(fileURLToPath(new URL('../public/icons/icon-512.png', import.meta.url)));
  await sharp(svgBuffer)
    .resize(410, 410)
    .extend({ top: 51, right: 51, bottom: 51, left: 51, background: NEWSprint })
    .png()
    .toFile(fileURLToPath(new URL('../public/icons/icon-maskable-512.png', import.meta.url)));

}

if (process.argv[1] === fileURLToPath(import.meta.url)) {
  await generate();
}
```
